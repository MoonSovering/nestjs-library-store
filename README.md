# Description
An REST API with [Nest](https://github.com/nestjs/nest) and [Mongo](https://www.mongodb.com/es)

## Installation

1. Clone repository
2. Execute

```bash
$ yarn install
```

3. Add NestCLI
```bash
$ yarn add -g @nestjs/cli
```

4. Clone file __.env.template__ and rename to __.env__

5. Fill the missing fields of the __.env__ file

6. Up database
```bash
docker-compse up -d
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
